Movie Rating Prodiction

In this project, my groupmates and I aimed to use machine learning algorithm to predict userâ€™s ratings on unseen movies based on data from various perspectives. The data used contain user ratings on seen movies, weights of each movie genre corresponding to each movie, and average movie ratings of each movie on major movie websites such as IMDB. We used the 10M MovieLens Dataset. We used an ensemble model of Linear Regression, Matrix Factorization, SVD, Gradient Boosting, kNN. Our goal was to make the prediction as close to real rating as possible, and we used RMSE (Root Mean Standard Error) to measure our performance. The smaller the RMSE, the more accurate our prediction is. We finally reached an RMSE of 0.822.

Ludo Game

This project was originated from Aeroplane Chess, a Chinese cross-and-circle board game similar to the western game of Ludo and the Indian game of Pachisi. Developed in the 20th century, Aeroplane Chess features airplanes as pieces instead of the more abstract pawns and beehive-shaped pieces found in the games from which it is derived.

Rule:

  1. Each player starts with 4 planes in the airport, and each player rolls dice alternatively.
  2. Each plane will not get out of the airport if the player does not roll a 6.
  3. Once the player rolls a 6, the player can choose a plane to get out of the airport.
  4. After that, the player can move the selected plane steps of the dice result.
  5. Once all four planes of a player get to the destination, the player win.
